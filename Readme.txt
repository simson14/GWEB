------------------------------------------------------------------------------------------------------------------------
GWEB : 개요
------------------------------------------------------------------------------------------------------------------------
    1.Web 기반의 SI Project를 진행할때마다 중복적으로 개발환경 구성에 많은 시간과 비용이 소모되므로 Project 초기
      개발환경 구성에 소요되는 시간과 비용을 절감하기 위하여 Maven 기반의 Spring Web Framework을 구현한다.

------------------------------------------------------------------------------------------------------------------------
GWEB : 주요 Library 구성 - 상세 내용은 pom.xml 확인요망
------------------------------------------------------------------------------------------------------------------------
    1. Servlet 3.0
    2. Spring 4.1.4
    3. MyBatis 3.2.8
    4. LogBack 1.1.2
    5. Sitemesh 3.0
    6. AspectJ 1.8.5
------------------------------------------------------------------------------------------------------------------------
GWEB : 주요 기능
------------------------------------------------------------------------------------------------------------------------
    1. AOP를 이용한 logging 기능 - Client의 요청을 처리 순서에 따라 Depth형으로 로깅
    2. AOP를 이용한 각 메소들별 처리시간 logging
    3. Auto Reloading Query - Mybatis 쿼리 변경시 서버의 restart 없이 변경쿼리를 적용(서비 기동후 신규 쿼리 추가는 제외)
    4. Auto Query Comment 자동추가 - DB에서 튜닝이 필요한 쿼리 추출시 해당쿼리가 어떤 쿼리인지 알수 있도록
                                     쿼리 Comment를 자동추가(Ex : SELECT /* test.xml[쿼리파일명]->aaa[쿼리ID] */)
    5. Config 파일 Auto Reloading - Configration 파일 변경시 자동 리로딩
    6. 선택적 Configuration 적용(Bean Definition Profile) - 서비 기동시 VM 옵션의 값에 따라 동일 Configuration 파일에서
          선택적으로 Configuration 적용
    7. Annotation을 이용한 Server Side Parameter Validation 체크
    8. Sitemesh를 이용한 공통 화면(Top/Left/Footer) 자동 구성
